# Convert a Return Table to a Brood Table

Convert a Return Table to a Brood Table

## Usage

``` r
return_to_brood(rt, by_stock = TRUE)
```

## Arguments

- rt:

  return table with column names: Stock, ReturnYear, and for each age
  (e.g., Age3). Must be in that format.

- by_stock:

  boolean for whether the data contians a Stock column.
