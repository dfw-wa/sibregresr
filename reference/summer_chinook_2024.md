# Summer chinook returns through 2024

Output of TAC summer Chinook managment season run reconstructions
through return to the Columbia River mouth year 2024

## Usage

``` r
summer_chinook_2024
```

## Format

### `summer_chinook_2024`

A tibble with 28 rows and 6 columns

- BroodYear:

  Brood year

- Age3, Age4, Age5, Age6:

  returns by age

- Stock:

  name of stock
